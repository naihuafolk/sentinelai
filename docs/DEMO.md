# สคริปต์สาธิต SentinelAI (Demo Script)

ใช้สำหรับนำเสนอลูกค้า/ผู้บริหาร ทีละขั้น ~5 นาที

## เตรียม (1 นาที)

```bash
cd backend
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```
เปิด http://127.0.0.1:8000 → กด **"โหลดข้อมูลตัวอย่าง"** (Dashboard มีข้อมูล 14 วันทันที)
ติดตั้ง extension จากโฟลเดอร์ `extension/` (ดู README)

---

## ฉาก 1 — "มองเห็น" (Visibility) · Dashboard

1. เปิดแท็บ **ภาพรวม** — ชี้ 4 ตัวเลข: เหตุการณ์ตรวจพบ, บล็อกการรั่วไหล, แผนกเสี่ยงสูงสุด, % เครื่องที่มี Agent
2. ชี้กราฟแนวโน้ม 14 วัน + สัดส่วนตามช่องทาง AI + ระดับความลับ
3. เลื่อนดู **เหตุการณ์ล่าสุด** — ใครพยายามส่งอะไรไป AI ตัวไหน ผลเป็นอย่างไร

> พูด: *"วันนี้องค์กรส่วนใหญ่ตาบอดเรื่องนี้ — SentinelAI ทำให้เห็นทุกความพยายามส่งข้อมูลลับออก AI"*

## ฉาก 2 — "ควบคุม" (Control) · ดักจับสด

เปิด **chatgpt.com** (หรือ Gemini/Claude/Copilot/DeepSeek) แล้วลองทีละอย่าง:

| ทำ | เกิดอะไร |
|---|---|
| วางข้อความ `API key: sk-proj-abcd1234efgh5678ijkl9012mnop3456` | ⛔ กล่อง **บล็อก** สีแดง — ไม่ให้ส่ง |
| วาง `เลขบัตรประชาชนลูกค้า 1101700207366 เบอร์ 0812345678` | 🛡️ ระบบ **ปิดบัง** ตัวเลขให้อัตโนมัติในช่องพิมพ์ |
| พิมพ์ `งบการเงินภายในยังไม่ประกาศ กำไรสุทธิ 214 ล้าน` แล้วกด Enter | ⚠️/⛔ **เตือน/บล็อก** พร้อมเหตุผล + คำแนะนำ (Coaching) |
| พิมพ์ `ขอสูตรต้มยำกุ้ง` แล้วกด Enter | ✅ **ผ่านปกติ** — ไม่รบกวนงานทั่วไป |

> พูด: *"ปกป้องโดยไม่ขวางการทำงาน — แทรกเฉพาะเมื่อเจอข้อมูลลับจริง"*

## ฉาก 3 — Fingerprint (จับของที่ถูกลอก)

1. Dashboard → แท็บ **เอกสารลับ** → ลงทะเบียนย่อหน้าจากสัญญา/งบลับ (วางข้อความ)
2. กลับไปเว็บ AI → วาง **แค่บางส่วน** ของเอกสารนั้น (คัดลอกไม่หมด/ดัดแปลงเล็กน้อย)
3. ระบบยังจับได้ → พิสูจน์ว่าจับ "การคัดลอกบางส่วน" ได้

## ฉาก 4 — "พิสูจน์ได้" (Compliance) + AI

1. แท็บ **เหตุการณ์** → กด **ส่งออก CSV** (หลักฐานสำหรับ PDPA/ISO 27001)
2. แท็บ **นโยบาย** → โชว์กฎ เช่น *"Restricted + เว็บ AI สาธารณะ → บล็อก"*, *"PII ทุกกรณี → ปิดบัง"* แล้วลองสร้าง/แก้กฎสด
3. แท็บ **ทดสอบระบบ** → วางข้อความ กดปุ่ม **ตัวอย่าง** ต่าง ๆ ดูคะแนนความเสี่ยง/เหตุผล
4. แท็บ **ระบบ & โมเดล** → โชว์ว่าใช้ **BytePlus ModelArk** (reasoning/vision/embedding)
   - ถ้าใส่คีย์แล้ว: เปิด AI แล้วทดสอบข้อความ **เชิงกลยุทธ์ที่ไม่มีรูปแบบตายตัว** (เช่น *"เราจะควบรวมกับบริษัท X ปีหน้า"*) → AI จับได้ทั้งที่ Regex จับไม่ได้
