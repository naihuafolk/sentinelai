SentinelAI - Endpoint Agent (ตัวป้องกันข้อมูลรั่วในคอมพิวเตอร์)
================================================================

ตัวนี้เฝ้าคลิปบอร์ดของเครื่อง เมื่อคัดลอกข้อมูลลับ (บัตรเครดิต,
บัตรประชาชน, รหัส/คีย์ ฯลฯ) แล้วจะเอาไปวางในแอป AI/แชท/อีเมล
ระบบจะเตือนหรือบล็อกให้อัตโนมัติ ก่อนข้อมูลจะหลุดออกไป


วิธีติดตั้ง (ง่ายมาก 3 ขั้น)
----------------------------------------------------------------
1) ให้เครื่องมี Python ก่อน
   - ถ้ายังไม่มี ดาวน์โหลดฟรีที่: https://www.python.org/downloads/
   - ** ตอนติดตั้ง Python ให้ติ๊กช่อง "Add Python to PATH" ด้วย **

2) ดับเบิลคลิกไฟล์   install.bat
   - โปรแกรมจะเตรียมทุกอย่างให้เอง
   - พอถามหา "Org Key" ให้วางคีย์ที่คัดลอกมาจากเว็บ
     (เข้า https://sentinelai.help -> ล็อกอิน -> แท็บ "ตั้งค่า"
      -> กดคัดลอก Org Key ขององค์กร)

3) เสร็จ! โปรแกรมจะเฝ้าให้อัตโนมัติ และเปิดเองทุกครั้งที่เปิดเครื่อง


ทดสอบว่าทำงาน
----------------------------------------------------------------
ลองคัดลอก (Ctrl+C) เลขบัตรเครดิตหรือเลขบัตรประชาชน
-> จะมีหน้าต่างของ SentinelAI เด้งเตือน/บล็อกขึ้นมามุมจอ


ถอนการติดตั้ง
----------------------------------------------------------------
ดับเบิลคลิกไฟล์   uninstall.bat


ต้องการความช่วยเหลือ: https://sentinelai.help
