1.บทนำและการยอมรับข้อกำหนด

ข้อกำหนดการใช้บริการฉบับนี้ (“ข้อกำหนด”) เป็นข้อตกลงระหว่างท่าน ในฐานะองค์กรผู้ใช้บริการ (“ลูกค้า” หรือ “ท่าน”) กับผู้ให้บริการ SentinelAI (“SentinelAI”, “เรา”) ในการเข้าถึงและใช้งานบริการ SentinelAI ผ่านเว็บไซต์ https://sentinelai.help รวมถึงซอฟต์แวร์ ส่วนขยาย (agent/extension) และบริการที่เกี่ยวข้อง

SentinelAI เป็นระบบป้องกันข้อมูลลับรั่วไหล (AI Data Loss Prevention — AI-DLP) ที่ช่วยลดความเสี่ยงในการที่ ข้อมูลลับขององค์กรจะถูกส่งเข้าสู่เครื่องมือ AI สาธารณะ เช่น ChatGPT, Gemini, Claude และ Copilot

การสร้างบัญชี การเข้าสู่ระบบ หรือการใช้บริการไม่ว่าส่วนใดส่วนหนึ่ง ถือว่าท่านได้อ่าน เข้าใจ และ ยอมรับที่จะผูกพันตามข้อกำหนดนี้ รวมถึง นโยบายความเป็นส่วนตัว ที่ถือเป็นส่วนหนึ่งของข้อกำหนดนี้ หากท่านไม่ยอมรับ โปรดหยุดใช้บริการ ผู้ที่ยอมรับข้อกำหนดในนามองค์กรรับรองว่าตนมีอำนาจผูกพันองค์กรนั้น

2.คำนิยาม

  • บริการ (Service): แพลตฟอร์ม SentinelAI ทั้งหมด รวมถึงคอนโซลบริหารจัดการ (dashboard), agent/ส่วนขยายที่ติดตั้งบนอุปกรณ์, API และเอกสารประกอบ
  • องค์กร / ลูกค้า (Organization / Customer): นิติบุคคลหรือบุคคลที่สมัครและชำระค่าบริการ และเป็นผู้กำหนดนโยบายการใช้งานภายในองค์กรของตน
  • ผู้ใช้ (User): พนักงานหรือบุคคลที่องค์กรอนุญาตให้ใช้งานบริการภายใต้บัญชีขององค์กร
  • Org Key: รหัสประจำองค์กรที่ใช้เชื่อมต่อ agent บนอุปกรณ์เข้ากับบัญชีองค์กรของท่าน ถือเป็นข้อมูลลับที่ต้องเก็บรักษา
  • สิทธิ์ต่อเครื่อง (Seat): สิทธิ์การใช้งานหนึ่งหน่วย ซึ่งอนุญาตให้ติดตั้งและใช้งาน agent ได้ 1 เครื่อง (device) ต่อ 1 สิทธิ์
  • เหตุการณ์ (Event): บันทึกการตรวจพบและการตัดสินของระบบเมื่อผู้ใช้พยายามส่งข้อความ/ไฟล์ไปยังเครื่องมือ AI
  • เนื้อหา (Content): ข้อความ ไฟล์ หรือข้อมูลที่ผู้ใช้พิมพ์หรือส่งผ่านช่องทางที่ระบบตรวจสอบ

3.บัญชีและการสมัครใช้งาน

ท่านต้องลงทะเบียนด้วยข้อมูลที่ถูกต้อง ครบถ้วน และเป็นปัจจุบัน อย่างน้อยได้แก่อีเมลและชื่อองค์กร ท่านมีหน้าที่รักษาความลับของข้อมูลเข้าสู่ระบบ (รหัสผ่าน และ Org Key) และรับผิดชอบต่อกิจกรรมทั้งหมด ที่เกิดขึ้นภายใต้บัญชีของท่าน

  • ท่านต้องแจ้งเราทันทีหากพบการเข้าถึงบัญชีโดยไม่ได้รับอนุญาต หรือสงสัยว่า Org Key รั่วไหล
  • ผู้สมัครต้องมีอายุและสถานะทางกฎหมายเพียงพอที่จะทำสัญญาผูกพันในนามองค์กร
  • บัญชีหนึ่งใช้สำหรับหนึ่งองค์กร ห้ามโอนบัญชีให้ผู้อื่นโดยไม่ได้รับความยินยอมเป็นลายลักษณ์อักษรจากเรา

4.การทดลองใช้ฟรี 14 วัน

เราอาจเสนอ ระยะทดลองใช้งานฟรีเป็นเวลา 14 วัน เพื่อให้ท่านประเมินบริการ โดยมีเงื่อนไขดังนี้

  • ระยะทดลองใช้เริ่มนับตั้งแต่วันที่เปิดใช้งานบัญชีองค์กร
  • เมื่อสิ้นสุดระยะทดลองใช้ หากท่านไม่เลือกแพ็กเกจแบบชำระเงิน บริการบางส่วนหรือทั้งหมดอาจถูกจำกัดหรือระงับ
  • เราขอสงวนสิทธิ์ในการปรับเงื่อนไข ระยะเวลา หรือยกเลิกข้อเสนอทดลองใช้ได้ตามดุลพินิจ
  • ระยะทดลองใช้มีไว้เพื่อการประเมินการใช้งานตามปกติเท่านั้น ไม่ใช่เพื่อการใช้งานเชิงพาณิชย์เต็มรูปแบบ

5.การชำระเงินและแพ็กเกจ

บริการคิดค่าใช้จ่ายในลักษณะ รายเครื่องต่อเดือน (per-device per month) ตามจำนวนสิทธิ์ (seats) ที่ท่านซื้อ ราคาที่แสดง ณ เว็บไซต์เป็นราคาที่ใช้บังคับ (อาจไม่รวมภาษีมูลค่าเพิ่มหรือภาษีอื่นที่กฎหมายกำหนด เว้นแต่ระบุไว้เป็นอย่างอื่น)

5.1 การเรียกเก็บและการต่ออายุ

  • ค่าบริการเรียกเก็บล่วงหน้าเป็นรายรอบ (รายเดือน) ตามจำนวนสิทธิ์ที่ใช้งานอยู่
  • การสมัครสมาชิกจะ ต่ออายุอัตโนมัติ ในรอบถัดไป เว้นแต่ท่านจะยกเลิกก่อนสิ้นสุดรอบปัจจุบัน
  • หากท่านเพิ่มจำนวนสิทธิ์ระหว่างรอบ อาจมีการคิดค่าบริการตามสัดส่วน (pro-rata) ของรอบนั้น
  • การชำระเงินล่าช้าหรือไม่สำเร็จอาจส่งผลให้บริการถูกจำกัดหรือระงับจนกว่าจะชำระครบถ้วน

5.2 การคืนเงิน

โดยหลักการ ค่าบริการที่ชำระแล้วสำหรับรอบที่ใช้งานไปแล้ว ไม่สามารถขอคืนได้ เว้นแต่กรณีที่กฎหมายกำหนด หรือเป็นไปตามนโยบายคืนเงินเฉพาะที่เราประกาศไว้เป็นลายลักษณ์อักษร การยกเลิกจะมีผลเมื่อสิ้นสุดรอบการเรียกเก็บที่ชำระไว้แล้ว โดยท่านยังใช้งานได้จนสิ้นรอบนั้น

6.สิทธิ์การใช้งาน (Seat / ต่อเครื่อง)

เราให้สิทธิ์การใช้งานแบบ ไม่ผูกขาด โอนสิทธิ์ไม่ได้ และเพิกถอนได้ เพื่อใช้บริการภายในองค์กรของท่าน ตามจำนวนสิทธิ์ที่ชำระค่าบริการ ภายใต้เงื่อนไขสำคัญดังนี้

  • 1 สิทธิ์ = 1 เครื่อง (device): แต่ละสิทธิ์อนุญาตให้ใช้งาน agent ได้บนอุปกรณ์เดียว
  • ห้ามแชร์ Org Key เกินจำนวนสิทธิ์ที่ซื้อ: การนำ Org Key ไปติดตั้งบนอุปกรณ์มากกว่าจำนวนสิทธิ์ที่ชำระ ถือเป็นการละเมิดข้อกำหนด
  • เพื่อบังคับสิทธิ์นี้ ระบบอาจใช้ลายนิ้วมืออุปกรณ์ (device fingerprint) และหมายเลข IP เพื่อระบุจำนวนอุปกรณ์ที่ใช้งานจริง (ดูรายละเอียดใน นโยบายความเป็นส่วนตัว)
ผลของการละเมิด: หากตรวจพบการใช้งานเกินจำนวนสิทธิ์ หรือการแชร์ Org Key เกินขอบเขต เราขอสงวนสิทธิ์ในการ ระงับ (suspend) หรือบล็อก อุปกรณ์/บัญชีที่เกี่ยวข้อง และ/หรือเรียกเก็บค่าบริการเพิ่มเติม ตามจำนวนอุปกรณ์ที่ใช้งานจริง

7.ข้อจำกัดการใช้งานและพฤติกรรมต้องห้าม

ท่านและผู้ใช้ของท่านตกลงว่าจะไม่กระทำการดังต่อไปนี้

  • ทำวิศวกรรมย้อนกลับ ถอดรหัส คัดลอก หรือพยายามเข้าถึงซอร์สโค้ดของบริการโดยไม่ได้รับอนุญาต
  • ใช้บริการเพื่อวัตถุประสงค์ที่ผิดกฎหมาย หรือละเมิดสิทธิของบุคคลอื่น
  • พยายามหลบเลี่ยง รบกวน หรือทำให้มาตรการรักษาความปลอดภัยหรือการจำกัดสิทธิ์ของระบบเสียหาย
  • แชร์หรือจำหน่ายต่อซึ่งสิทธิ์การใช้งาน หรือ Org Key เกินขอบเขตที่ได้รับอนุญาต
  • ใช้บริการในลักษณะที่สร้างภาระเกินสมควรต่อโครงสร้างพื้นฐานของเรา

8.ข้อจำกัดความรับผิดและการรับประกัน

บริการนี้ให้ “ตามสภาพที่เป็นอยู่” (as is) และ “ตามที่มีให้ใช้ได้” (as available) เท่าที่กฎหมายอนุญาต เราไม่รับประกันโดยชัดแจ้งหรือโดยปริยายว่าบริการจะปราศจากข้อผิดพลาด ทำงานต่อเนื่องไม่หยุด หรือเหมาะสมกับวัตถุประสงค์เฉพาะใด ๆ

SentinelAI เป็น เครื่องมือช่วยลดความเสี่ยง (risk-reduction tool) ในการป้องกันข้อมูลลับรั่วไหลสู่ AI สาธารณะ ไม่ใช่การรับประกันความปลอดภัยแบบ 100% ระบบอาจไม่สามารถตรวจจับหรือยับยั้งการรั่วไหลได้ในทุกกรณี (เช่น ช่องทางที่ไม่ครอบคลุม, การถ่ายภาพหน้าจอ, การพิมพ์ซ้ำด้วยมือ ฯลฯ)
  • ลูกค้ายังคงเป็นผู้รับผิดชอบข้อมูลของตน รวมถึงการกำหนดนโยบายภายใน การอบรมพนักงาน และการปฏิบัติตามกฎหมายที่เกี่ยวข้อง
  • เท่าที่กฎหมายอนุญาต เราไม่รับผิดต่อความเสียหายทางอ้อม ความเสียหายพิเศษ ผลสืบเนื่อง การสูญเสียกำไร ข้อมูล หรือชื่อเสียง
  • ความรับผิดรวมสูงสุดของเราต่อความเสียหายใด ๆ จะไม่เกินจำนวนค่าบริการที่ท่านชำระจริงให้แก่เราในช่วง 12 เดือน ก่อนเกิดเหตุ

9.ทรัพย์สินทางปัญญา

บริการ ซอฟต์แวร์ โลโก้ เครื่องหมายการค้า การออกแบบ และเนื้อหาทั้งหมดของ SentinelAI เป็นทรัพย์สินของเราหรือผู้ให้สิทธิ์ของเรา และได้รับความคุ้มครองตามกฎหมายทรัพย์สินทางปัญญา ข้อกำหนดนี้ไม่ได้โอนกรรมสิทธิ์ใด ๆ ให้แก่ท่าน นอกจากสิทธิ์การใช้งานที่ระบุไว้

ข้อมูลขององค์กร (Customer Data) ยังคงเป็นของท่าน เราใช้ข้อมูลดังกล่าวเพียงเพื่อให้บริการตามข้อกำหนดและ นโยบายความเป็นส่วนตัว เท่านั้น

10.ข้อมูลส่วนบุคคลและความเป็นส่วนตัว

การเก็บ ใช้ และประมวลผลข้อมูลส่วนบุคคลเป็นไปตาม นโยบายความเป็นส่วนตัว ของเรา ซึ่งจัดทำโดยคำนึงถึง พระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA) ในความสัมพันธ์นี้ โดยทั่วไป ลูกค้าองค์กรเป็นผู้ควบคุมข้อมูล (Data Controller) และ SentinelAI เป็นผู้ประมวลผลข้อมูล (Data Processor)

11.การระงับและการยกเลิกบริการ

  • ท่านสามารถยกเลิกการสมัครสมาชิกได้ทุกเมื่อ โดยการยกเลิกจะมีผลเมื่อสิ้นสุดรอบการเรียกเก็บที่ชำระไว้แล้ว
  • เราอาจระงับหรือยกเลิกบริการทั้งหมดหรือบางส่วน หากท่านละเมิดข้อกำหนด ค้างชำระค่าบริการ หรือใช้งานในลักษณะที่ก่อความเสี่ยงด้านความปลอดภัยหรือกฎหมาย
  • เมื่อบริการสิ้นสุดลง สิทธิ์การใช้งานจะยุติทันที ท่านควรส่งออก (export) ข้อมูลที่จำเป็นก่อนสิ้นสุดบริการ เราจะจัดการข้อมูลตามที่ระบุใน นโยบายความเป็นส่วนตัว

12.การเปลี่ยนแปลงข้อกำหนด

เราอาจปรับปรุงข้อกำหนดนี้เป็นครั้งคราว หากมีการเปลี่ยนแปลงในสาระสำคัญ เราจะแจ้งให้ทราบผ่านช่องทางที่เหมาะสม (เช่น อีเมล หรือประกาศบนเว็บไซต์) และปรับปรุงวันที่ “มีผลบังคับใช้” ด้านบน การใช้บริการต่อไปหลังการเปลี่ยนแปลงมีผล ถือว่าท่านยอมรับข้อกำหนดฉบับปรับปรุง

13.กฎหมายที่ใช้บังคับ

ข้อกำหนดนี้อยู่ภายใต้บังคับและการตีความตาม กฎหมายแห่งราชอาณาจักรไทย ข้อพิพาทใด ๆ ที่เกิดขึ้นจากหรือเกี่ยวข้องกับ ข้อกำหนดนี้ให้อยู่ในเขตอำนาจของศาลไทย เว้นแต่กฎหมายจะกำหนดเป็นอย่างอื่น
